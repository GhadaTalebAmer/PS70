Converting a bitmap picture into a vector format

I began by trying the example on the site. I followed the steps to trace the bitmap of the paddle sketch picture into a vector. I played around with the features in the dialog box to see how the picture would change. I changed the line color to red as shown below.

Steps followed:
  1. Import bitmap of paddle sketch into Inkspace sheet
  2. Traced bitmap with edge detection
  3. Deleted the bitmap, left me with vectors (looks like outline of the image)
  4. Change style with fill = no colour and stroke = red
  5. Broke apart the paths and removed the extra lines by deleting them
  6. Kept simplifying the path until smooth
  7. Exported the image as a png to add to my website
